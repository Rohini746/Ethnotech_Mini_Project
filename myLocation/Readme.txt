# 📍 My Location Web App

## 📌 Project Overview
The **My Location Web App** is a simple web application built using **HTML, CSS, and JavaScript**.  
This project allows the user to turn on their device location and find their **current location**.  
When the user gives permission, the application accesses the browser’s geolocation feature and displays the user’s current position.

---

## 🚀 Features
- 📍 Detects the user’s current location  
- ✅ Requests location permission from the user  
- 🌐 Uses browser geolocation API  
- 📌 Displays latitude and longitude  
- ⚡ Simple and user-friendly interface  

---

## 🛠️ Technologies Used
- **HTML** – for creating the structure of the webpage  
- **CSS** – for styling the webpage  
- **JavaScript** – for adding functionality  
- **Geolocation API** – for fetching the current location of the user

▶️ How to Run the Project
Download or open the project folder
Open the index.html file in a browser
Click on the button to allow location access
Grant permission when the browser asks
The app will show the current location of the user

💡 How It Works
The user opens the webpage
The user clicks the location button
The browser asks for location permission
After permission is granted, JavaScript uses the Geolocation API
The app fetches the current coordinates of the user
The location details are displayed on the webpage

⚠️ Requirements
Internet connection may be needed in some browsers
Location permission must be enabled
The browser should support Geolocation API

❌ Error Handling
Shows an error if the user denies location access
Shows a message if the browser does not support geolocation
Handles cases when location cannot be fetched
🎯 Future Enhancements
🗺️ Show location on Google Maps
📍 Display full address using reverse geocoding
🌎 Add live location tracking
📱 Make the UI more responsive and attractive

🙌 Conclusion

This project helps in understanding how to use the Geolocation API in JavaScript.
It is useful for beginners to learn how web applications can access device location with user permission.
